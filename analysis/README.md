# Business Analysis — Web Agency Platform

This folder contains the completed 2025-2026 business analysis for the AI-agent-operated web agency.

## Purpose

Separate from the operational docs in `docs/` and the execution control in `plans/`, this folder holds research-driven analysis:

- market sizing,
- competitive landscape,
- city + niche selection,
- pricing validation,
- technology and regulatory risk,
- financial projections,
- and action recommendations.

## Files

| File | What it contains |
|---|---|
| `01-EXECUTIVE-SUMMARY.md` | Top-level findings and recommended actions |
| `02-MARKET-ANALYSIS.md` | TAM / niche counts / website gap / market trends |
| `03-COMPETITIVE-LANDSCAPE.md` | Competitors, pricing, positioning gaps |
| `04-ICP-AND-NICHE-ANALYSIS.md` | Recommended city+niche pairs for first outreach |
| `05-PRICING-STRATEGY.md` | Market pricing benchmarks and pricing recommendations |
| `06-SWOT-ANALYSIS.md` | Strengths, weaknesses, opportunities, threats |
| `07-TECHNOLOGY-TRENDS.md` | Search, AI builder, scraping, and regulatory risks |
| `08-FINANCIAL-PROJECTIONS.md` | Funnel, CAC, LTV, and MRR path |
| `09-ACTION-PLAN.md` | 30-day, 90-day, and critical next decisions |

## Status

Completed on `2026-04-26` using current 2025-2026 web research. Where public data lagged or category definitions were inconsistent, estimates are labeled as estimates or inferences rather than presented as hard facts.
