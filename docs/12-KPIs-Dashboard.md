# KPIs and Dashboard

## Objective

Measure the business like a system. The first goal is not vanity metrics; it is proving that the AI-agent-operated model converts, delivers, and retains profitably with minimal human intervention.

## Weekly sales KPIs

Track weekly:

- Leads reviewed by agent
- Qualified leads
- Hot leads identified
- Hook emails sent
- Reply rate
- Positive reply rate
- Demos built (only for interested prospects)
- Demo-to-close conversion rate
- Payments collected
- Clients closed
- Phone-fallback leads (no email, has phone)
- Phone/SMS opener sent
- Phone/SMS reply rate

## Funnel math

Use this chain:

Leads reviewed → qualified leads → hook emails sent → replies → positive replies → demos sent → payments → active clients

## Agent quality KPIs

These are the most important metrics early on. If the agent produces poor output, nothing else matters.

- Agent email quality score (human-rated sample of 5–10 emails/week during validation)
- Agent audit accuracy (human spot-checks 5 audits/week)
- Agent demo quality score (human rates every demo during validation)
- Agent error rate (incorrect info, wrong client data, broken links)
- Spam complaint rate on outbound emails
- Agent hallucination incidents (promises made outside scope, wrong pricing, invented features)

## Delivery KPIs

- Average time to launch
- Human review time per launch
- Average number of revisions per client
- Included vs add-on task ratio
- Support ticket volume
- Agent-resolved vs human-resolved support tickets
- Critical issue count

## Financial KPIs

- Monthly recurring revenue
- Net revenue after fees
- Average revenue per client
- AI agent API costs per client per month
- Hosting and infrastructure cost per active client
- Gross margin per client
- Gross churn
- Net client growth

## Retention KPIs

- Month-1 retention
- Month-2 retention
- Month-3 retention
- Failed payment rate
- Cancellation reasons by category
- Percentage of clients receiving monthly value reports
- Churn signals detected by agent vs caught late

## Agent operational KPIs

- Agent uptime / availability
- Agent tasks completed per day
- Agent tasks requiring human intervention
- Average agent response time (support requests)
- Agent token usage per day / per client
- Agent cost as % of revenue (target: <10%)

## Early-stage benchmarks

The business should reach internal confidence when:

- The agent's prospecting list consistently produces obviously weak sites
- Agent-written hook emails are getting real replies (target: >5% reply rate)
- Agent-built demos are converting interested prospects (target: >30% demo-to-close)
- Launches are happening without chaos (target: <30 min human review per launch)
- Clients remain after the first renewal cycle
- Agent error rate is below 5% on customer-facing interactions

## Weekly review questions

- Which niche or city had the best reply rate?
- Which issue angle got the best response in hook emails?
- How do phone/SMS fallback reply rates compare to email reply rates?
- How many demos were requested vs how many emails sent?
- Which package is easiest to close?
- Did the agent make any errors this week? What kind?
- How much human time was spent on agent corrections?
- Are agent API costs per client stable or rising?
- Which client requests keep repeating and should become a productized feature?
- Is the 2-step sales motion working (hook → demo request rate → close rate)?
