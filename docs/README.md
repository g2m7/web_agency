# Business Documentation Pack

This pack captures the full operating system for an AI-agent-operated website-retainer agency. An AI agent runs the business 24/7 — prospecting, auditing, outreach, demos, onboarding, delivery, support, and billing. A human operator handles only what the agent cannot do: payment setup, legal decisions, final quality checks, and strategic direction.

## Core model

1. Agent identifies local businesses with weak websites in a chosen niche and geography.
2. Agent audits their site and scores it.
3. Agent sends a hook email highlighting one visible issue — no demo, just a question: "Want to see a better version?"
4. If the prospect replies interested, agent builds a tailored demo and sends it with pricing.
5. Agent closes the client onto a monthly subscription via hosted checkout (Dodo Payments).
6. Agent delivers hosting, managed updates, analytics, an AI chat assistant, and support.
7. Human operator reviews quality, handles escalations, and makes strategic decisions.

## How to use this pack

1. Read `01` through `03` first to align on strategy and agent operating model.
2. Read `06` carefully — the 2-step sales motion is the core change from a traditional agency.
3. Use `04` through `10` as the agent's operating handbook.
4. Use `11` to set up records from day one.
5. Review `12` weekly — agent quality KPIs are the most important metrics early on.
6. Review `13` monthly — agent-related risks are the top concern.
7. Treat `14` as the execution sequence — phases include human approval gates that are removed as agent quality proves out.
8. `15` contains all templates the agent uses for customer communication.
9. `16` contains the full business model audit and financial projections.

## Key differences from a traditional web agency

| Traditional agency | This model |
|---|---|
| Human does everything | AI agent operates 24/7 |
| Demo for every prospect | Demo only for interested prospects |
| Limited by human capacity | Scales with compute |
| High marginal cost per client | Near-zero marginal cost per client |
| Linear revenue growth | Compounding revenue with fixed opex |

## Intended outcome

Validate that the AI agent can produce customer-facing output that converts, then scale to 10–50+ clients with the human operator spending 3–10 hours/week on oversight and strategy.
